# 会员制度上线检查清单

## 一、环境准备

### 1. 后端环境
- [ ] `.env` 文件已配置
- [ ] `ENABLE_PAYMENT=true`（正式上线时）
- [ ] 微信支付商户参数已配置
- [ ] 证书文件已上传至服务器
- [ ] 支付回调域名已备案并配置 HTTPS

### 2. 小程序后台
- [ ] 小程序主体认证（企业）
- [ ] 服务器域名已配置（request/uploadFile）
- [ ] 业务域名已配置

## 二、功能验证

### 1. 试用期
- [ ] 新用户注册后自动激活15天试用
- [ ] 试用期功能正常（AI对话、评估、推荐）
- [ ] 试用到期后自动降级至免费版

### 2. 会员购买
- [ ] 月卡/季卡/年卡展示正确
- [ ] 创建订单成功
- [ ] 微信支付调起正常
- [ ] 支付成功后会员激活
- [ ] 支付回调处理正确

### 3. 自动续费
- [ ] 购买时默认勾选自动续费
- [ ] 用户可手动取消自动续费
- [ ] 到期前3天发送提醒
- [ ] 到期自动扣款成功

### 4. 兑换码
- [ ] 兑换码生成正常
- [ ] 兑换码兑换成功
- [ ] 过期/无效码处理正确

### 5. 裂变
- [ ] 邀请码生成正常
- [ ] 邀请关系记录正确
- [ ] 被邀请人购买后奖励发放
- [ ] 每月上限60天限制生效

## 三、安全验证

- [ ] API密钥未泄露
- [ ] 证书文件权限正确（600）
- [ ] 支付回调已验签
- [ ] 订单幂等性处理
- [ ] 防止重复支付

## 四、监控与告警

- [ ] 支付成功率监控
- [ ] 异常订单告警
- [ ] 数据库备份正常
- [ ] 日志记录完整

## 五、回滚方案

```bash
# 执行备份
bash /workspace/backend/scripts/backup-before-membership.sh

# 回滚（如需要）
bash /workspace/backend/scripts/rollback-membership.sh <备份路径>
```
